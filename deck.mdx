# Arrange 🤔

Layout and setup all variables and mocks required to run the test.

<img
  srcset="https://user-images.githubusercontent.com/6516758/48911627-38ead880-ee41-11e8-9434-6f925cf6b59a.png"

```notes
describe('Add', () => {
  it('should be able to add two numbers', () => {
    // Arrange

    const secondNumber = 2;
  });
});
```

---

# Act 🤺

Run the unit you are trying to test

<img
  srcset="https://user-images.githubusercontent.com/6516758/48911625-38ead880-ee41-11e8-8d5e-bd9d1d422642.png"
  src="/assets/act.png"
/>

```notes
describe('Add', () => {
  it('should be able to add two numbers', () => {
    // Arrange
    const firstNumber = 1;
    const secondNumber = 2;

    // Act
    const result = add(firstNumber, secondNumber);
  });
});
```

---

# Assert 🕵️‍♂️

Make sure that you get the result you were looking for! Jest and other test runners have built in assertions.

<img
  width="50%"
  srcset="https://user-images.githubusercontent.com/6516758/48911628-39836f00-ee41-11e8-9081-50c07c2f0003.png"
  src="/assets/assert.png"
/>

```notes
describe('Add', () => {
  it('should be able to add two numbers', () => {
    // Arrange
    const firstNumber = 1;
    const secondNumber = 2;

    // Act
    const result = add(firstNumber, secondNumber);

    // Assert
    expect(result).toBe(3);
  });
});
```
